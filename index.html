Dark Mode Demo

This is a simple project to test dark mode using JavaScript. Switch between light and dark to see how the colors adapt.

Why Dark Mode?

Dark mode reduces eye strain in low light. It also saves battery on OLED screens and looks sleek for many users.

Features of This Page

You can toggle between themes with a button. Text, background, and accent colors change automatically.

Try It Yourself

Click the toggle button above to switch between light and dark. Notice how headings, paragraphs, and other elements respond.
